CORE EXPERIENCE:
This is not a normal skills list. It should feel like entering a living digital universe of technologies.

1. TIME MACHINE MODE (Scroll-Based Evolution)
As the user scrolls through the section, the tech world evolves through eras:

ERA 1 — FOUNDATION (2018)
Only HTML, CSS, JavaScript appear.
Visual style: vintage, subtle grain texture, old-tech atmosphere.

ERA 2 — FRAMEWORK AGE (2020)
React, Vue, Angular emerge.
Background becomes cleaner, more modern and energetic.

ERA 3 — FULLSTACK ERA (2022)
Next.js, Laravel, Node.js, Docker appear.
Scene becomes deeper, more advanced, more structured.

ERA 4 — FUTURE ERA (2026)
AI, motion systems, realtime technologies.
Background transforms into neon futuristic cyber atmosphere with glowing effects.

Transitions between eras must be smooth, cinematic, and scroll-controlled.

2. USER INTERACTIVE MAGNETIC MODE
Mouse movement influences all floating icons.

Behavior:
- Icons near cursor are gently attracted.
- Icons farther away are softly pushed aside.
- When cursor hovers precisely over an icon, it snaps slightly toward the cursor.
- Motion should feel smooth, premium, physics-based, never chaotic.

Goal:
User feels like touching and controlling a digital constellation.

3. CARD BREAKS THE RULES (Main Hover Feature)
When hovering a technology icon (example: React):

- All other icons blur slightly and move backward.
- The selected icon zooms forward dramatically.
- A holographic skill card emerges from the icon.
- Card has subtle 3D rotation following mouse movement.
- Blue particle energy activates in background.
- On mouse leave, the card disintegrates into particles and returns into the icon.

Example card content:

React
3+ Years Experience
Projects: 12
UI Systems / State Management / Animation

Repeatable for all technologies with custom data.

4. POWER LEVEL SYSTEM
When hovering any stack icon:

- A glowing circular energy ring appears around the icon.
- Ring animates to display mastery percentage.

Examples:
React = 92%
CSS = 95%
Python = 70%

The ring should feel like an advanced scanner / power meter, not a normal progress bar.
